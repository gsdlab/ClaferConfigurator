Instance Generator

In this window, you can run and control the instance (configuration) generator.
  • Use the button "" to begin a session with the selected instance generator.
  • Use the button "" to kill a session with a running instance generator.
  • Use the button "" to ask for next N instances (if possible).
  • Use the button "" to reset the settings made to the current instance generator session.
  • Use the button "" to finish the session safely.
  • Adjust scopes or Maximum Integer bound settings using "" or "" buttons.
