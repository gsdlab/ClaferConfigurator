Welcome to ClaferConfigurator

In this tool, you can explore the configurations of an "attributed feature model with inheritance" subset of Clafer.

Press the '?' button of each window to see specific help. In general:

-- Use the input view to select a .cfr model and generate configurations.

-- Use the Control window to set the number of instances to be produced within the given scope.

-- Use the output view to see if there are no more instances or if the model contains an unsat core.

-- Use the Feature and Quality Matrix View to filter the instances by selecting and eliminating features.
