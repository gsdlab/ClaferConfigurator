In this tool, you can edit, compile, explore, and configure attributed feature models with inheritance expressed using Clafer. For guidelines, see the wiki page Guidelines for feature modeling using Clafer.
Press the '?' button of each window to see specific help. In general:
  • Use the view Input Clafe Model and Options to edit and compile a model.
  • Use the view Instance Generator to run an instance generator and adjust the bitwidth.
  • Use the view Feature and Quality Matrix to filter the configurations by selecting and eliminating features.
  • Use the view Constraints to view and copy
  • Use the view Output to see the messages and UnSAT cores.
