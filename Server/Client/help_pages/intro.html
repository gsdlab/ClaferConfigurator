In this tool, you can edit, compile, explore, and instantiate Clafer models. Press the '?' button of each window to see specific help. In general:
  • Use the Input File or Example view to compile and instantiate a model.
  • Use the Clafer Source Model view to explore a HTML rendering of a model.
  • Use the Control view to generate more instances and adjust the scope.
  • Use the Output view to see instances, errors, and UnSAT cores.
